# Smart Travel Companion 🚀

A feature-rich Flutter travel application built with clean architecture, API integration, animations, offline caching, and state management.

## ✨ Features

* Fetch travel places from API
* Search with debounce and filters
* Pull-to-refresh and pagination
* Favorites functionality
* Hero animations and smooth UI transitions
* Weather integration using Open-Meteo API
* Offline caching support
* Dark mode support
* Maps integration
* Notifications
* Retry and empty-state handling

## 🛠 Tech Stack

* Flutter & Dart
* Riverpod / Bloc
* REST APIs
* GoRouter / Named Routes
* Hive / SharedPreferences / SQLite

## 📡 APIs Used

* JSONPlaceholder Photos API
* Open-Meteo Weather API

## 📂 Project Structure

lib/

* features/
* models/
* services/
* providers/

* widgets/

## ▶️ How to Run

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
